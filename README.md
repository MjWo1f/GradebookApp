# Electronic Gradebook System

## Overview

The Electronic Gradebook System is a role-based academic management application designed to help administrators, teachers, and students efficiently manage grades, assignments, and academic progress.

This project was developed using HTML, JavaScript, Electron, and database integration to simulate a real-world educational management platform with secure login functionality and interactive dashboards.

---

## Features

### Student Dashboard

* View grades and assignments
* Track academic progress
* GPA calculation system
* At-risk student highlighting
* Progress reports by class

### Teacher Dashboard

* Manage students and grades
* Generate progress reports
* Import student and grade data using CSV files
* View class performance analytics

### Admin Dashboard

* Add and manage teachers
* View teacher accounts
* Secure administrative controls

### Authentication & Security

* Login system with role-based access
* Password encryption using bcrypt
* Forced password change support
* Session management

## System Architecture

The system is divided into multiple role-based dashboards:

* `admin_dashboard.html`
* `teacher_dashboard.html`
* `student_dashboard.html`

Each dashboard provides unique functionality based on the authenticated user role.

---

## Key Functionalities

### GPA Calculation

The application automatically calculates GPA values based on student grade averages and displays academic progress reports.

### CSV Import System

Teachers can import student and assignment data through CSV files to streamline classroom management.

### Progress Monitoring

The system highlights students who are academically at risk by detecting averages below a specified threshold.
